# 📊 2024 Regional Sales & Strategic Analysis

A data-driven analysis of 2024 regional sales performance across the US. Featuring Python data processing and an interactive Power BI dashboard to identify high-margin products and growth opportunities.

## 🌟 Project Overview
This project provides a comprehensive analysis of sales performance across four major US regions (West, Midwest, Northeast, and South). By integrating sales orders, customer demographics, and 2024 budget data, this analysis identifies key revenue drivers and profitability trends to support data-driven business decisions.

The goal of this project is to transform raw transactional data into actionable strategic insights for regional managers and stakeholders.

## 🚀 Key Business Insights
* **Regional Growth:** Identified **California** and **Connecticut** as the highest revenue-generating hubs, with individual large-scale orders exceeding **$30,000**.
* **Profitability Gems:** Discovered that **Product 17** maintains a peak profit margin of **60%**, significantly outperforming the company average.
* **Budget Alignment:** Conducted a performance-to-budget analysis against the **$5.6M+** allocated for top-performing products like **Product 26**.
* **Channel Efficiency:** Evaluated the effectiveness of **Wholesale**, **Distributor**, and **Export** channels in different geographic territories.

## 🛠️ Tech Stack
* **Data Processing:** Python (Pandas, NumPy)
* **Visualization:** Power BI (Interactive Dashboards) & Matplotlib
* **Analysis:** Jupyter Notebooks for Exploratory Data Analysis (EDA)
* **Data Format:** Excel / CSV

## 📂 Project Structure
* `Analysis.ipynb`: Python notebook containing data cleaning, merging, and initial statistical analysis.
* `Regional Data Analysis.pbix`: Interactive Power BI dashboard file.
* `file.csv` & `Power BI csv.csv`: Cleaned and processed datasets ready for reporting.
* `Regional Sales.xlsx`: Original raw data including sales orders, budgets, and customer lists.

## 📈 Summary of Findings
1. **High-Value Markets:** The West region leads in total volume, while the Midwest shows the highest growth potential for specialized product lines.
2. **Product Trends:** High-budget items like **Product 26** drive consistent revenue, but lower-volume items like **Product 15** contribute massive single-order spikes (e.g., **$56k+** in Ohio).
3. **Strategy Recommendation:** Focus marketing spend on high-margin states (Oregon/Ohio) and optimize supply chain routes for the high-volume California market.
